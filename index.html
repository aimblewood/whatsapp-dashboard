🚫 Network Restriction Detected

Your network is blocking external CDN libraries required for the dashboard to function.

Common Causes:

  • Corporate Firewall - Company networks often block CDNs
  • Ad Blockers - uBlock Origin, AdBlock Plus, etc.
  • ISP Restrictions - Some ISPs block certain domains
  • VPN/Proxy - May interfere with CDN access

Quick Solutions:

  1. Disable ad blockers temporarily
  2. Try a different network (mobile hotspot)
  3. Contact IT department about CDN access
  4. Use manual connection method below
